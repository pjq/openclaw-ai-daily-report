# AI Daily Report - March 2, 2026

## 技术要点

1. **AI治理投入激增但仍不足**: 全球AI治理平台支出预计2026年达4.92亿美元，2030年将超10亿美元，但Gartner认为这仍不足以应对AI风险。75%的经济体将在本年代末出台AI法规。

2. **LLM新突破：工具使用能力**: 2026年LLM的重大突破不是记忆能力，而是工具使用能力。模型可以调用Python等工具来执行任务，而非仅靠猜测回答问题。

3. **mHC架构重新定义LLM训练**: 一种新的范式"流形约束超连接"(mHC)被提出，旨在突破标准Transformer和混合专家模型的效率极限。

4. **硅谷41万岗位面临AI重塑**: 硅谷指数报告显示，约41万个当地工作岗位将受到AI影响，AI正在改变就业结构。

5. **AI制药进入闭环时代**: 采用AI的生物技术公司中，50%已实现更快的靶点发现时间，42%在科学模型准确性上有所提升。AI设计系统与实验室的紧密耦合正在将药物发现时间从数年缩短至数月。
