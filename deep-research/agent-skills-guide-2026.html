🎯 什么是 Agent Skills？

Agent Skills 是模块化的能力包，本质是一个文件夹，包含 SKILL.md 文件（定义指令和元数据），以及可选的脚本、模板和参考资料。

核心理念：不再每次都写长提示词，而是创建一个自包含的技能文件夹，让 AI 代理在需要时自动发现并应用相关上下文。

技能文件夹结构

my-skill/ ├── SKILL.md # 必需：指令 + 元数据 ├── scripts/ # 可选：可执行代码 ├── references/ # 可选：参考文档 └── assets/ # 可选：模板、资源文件

⚡ 为什么 Agent Skills 是 2026 年的关键趋势？

  1. 解决 AI 的核心局限性：复杂重复性任务中的不一致性问题
  2. 减少幻觉：通过结构化指令强制执行最佳实践
  3. 可扩展性：将专业知识封装为可复用模块
  4. 渐进式披露：按需加载，保持上下文窗口清洁
开发者报告：使用技能后，常规工作效率提升 70-90%，能够专注于高价值创作。

🛠️ 如何创建自定义技能

步骤 1：创建目录

# 在工作区创建技能目录 mkdir -p ~/.openclaw/skills/my-team-conventions

步骤 2：编写 SKILL.md

--- name: my-team-conventions description: Enforces our team's coding standards --- # Team Coding Conventions ## Architecture - Use modular design - Separate business logic ## Git rules - Follow conventional commits - Require review before merging

步骤 3：AI 自动应用

创建后，AI 代理会自动在相关场景下激活这些规范，就像与一位熟悉团队标准的团队成员协作。

🔄 技能工作原理

技能使用渐进式披露架构，保持代理上下文窗口清洁：

  1. 发现：代理根据任务识别需要哪个技能
  2. 加载：按需加载技能指令（不一次性加载所有技能）
  3. 执行：结合技能指令和当前任务上下文执行
  4. 释放：完成任务后释放上下文
